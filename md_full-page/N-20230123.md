对于串联弹簧，k = k1 + k2 + ... + kn
题中弹簧可视为小弹簧串联，把其分割为若干个Δx
共有n个。k = Δx/l * k0
k0 = l/Δx * k
∴Δx段弹簧的重量拉取Δx段弹簧，伸长量为
Δg/k0 = x/l * Δx
∴l = Δg/k0 * Σ xi * Δxi | N->∞
= Δg/k0 * ∫ x dx
= Δg * l^2 / 2k0
