例153. 已知正项数列 {an}, 前n项和为Sn, 对任意n∈N*, 有 (an+2)/2 = √2Sn, 用数学归纳法证明 an = 4n - 2.

解：当n = 1时, (a1 + 2)/2 = √2a1, (a1 + 2)² = 8a1, a1 = 2 = 4 x 1 - 2
若命题在n = k时成立，下面证明n = k + 1时也成立
(ak+1 + 2)/2 = √2Sk+1 = √2(Sk + ak+1) = √2Sk + 2ak+1
∴ (ak+1 + 2)² = 8(4k - 2) + 8ak+1
ak+1² - 4ak+1 + 4 - 32k + 16 = 0
ak+1 - 2 = √32k - 16 = 4√2k - 1
ak+1 = 4(k + 1) - 2
