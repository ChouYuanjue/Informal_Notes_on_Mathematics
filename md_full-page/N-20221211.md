表示方式：
y', f'(x), dy/dx, df(x)/dx, d/dx y, d/dx f(x)

例1：f(x) = x²，求f'(x), f'(-1), f'(2)
解：f'(x) = ((x+Δx)² - x²) / Δx = (x² + 2xΔx + (Δx)² - x²) / Δx = (2xΔx + (Δx)²) / Δx = 2x
f'(-1) = -2, f'(2) = 4

例2：求y = √x 过点(1,1)的切线和法线
解：f'(x) = (√(x+Δx) - √x) / Δx = 1 / (√(x+Δx) + √x) = 1 / (2√x)
