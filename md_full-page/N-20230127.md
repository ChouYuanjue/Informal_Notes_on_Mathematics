n = 4:

Proof 由于 (x^(k) + x^(k-1) + ... + x^2 + x + 1)(x - 1) = x^(k+1) - 1, 所以 x^k + x^(k-1) + ... + x^2 + x + 1 的根为 {ζ, ζ^2, ..., ζ^k}.
