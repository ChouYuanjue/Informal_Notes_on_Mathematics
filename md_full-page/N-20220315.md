1. 设二次函数f(x) = ax² - bx - c (a < b < c) 的图象过点(1,0)和B(m, -a).
(1) 若函数的顶点坐标为 (1/4, 25/16), 求实数m的值;
(2) 若函数图象的对称轴为x = x₀, 求x₀的取值范围;
(3) 若对x ≥ k (k是与a, b, c无关的常数)时, 恒有f(x) - a ≤ C, 试求实数k的最小值.

解: 设f(x) 与x轴交于(m, 0)
∴ am² + bm - b = 0
am² = -b(m - 1)
-b/2a = m²/2m-2
由(2)知, -1/2 ≤ x₀ ≤ 0,
即 -1/2 < m²/2m-2 < 0
设 m²/2m-2 = x, m² - (2m - 2)x
∴ m² - 2mx + 2x = 0
m₁ = (2x - √(4x² - 8x))/2, m₂ = (2x + √(4x² - 8x))/2
∴ m₂ > m₁
令 m(x) = (2x - √(4x² - 8x))/2
dm/dx = 1 - (x - 2)/√(4x² - 8x)
(i) x₀ 为区间端点
x₀ = -1/2 或 x₀ = 0
(ii) x₀ 处 m'(x) 不存在,
即 1 - (x - 2)/√(4x² - 8x) 无意义
∴ x₀ ∈ [-1/2, 0], ∴ x₀ = 0
(iii) x₀ 处 m'(x) = 0
即 1 - (x - 2)/√(4x² - 8x) = 0
x₀ = 2 (增根) 或 x₀ = -2/3
∴ x₀ ∈ [-1/2, 0], 舍
∴ x₀ = 0 或 -1/2 时取得局部极值
∴ x₀ = 0, m = 0; x₀ = -1/2, m = (√5 - 1)/2
∴ m < (√5 - 1)/2, k = (√5 - 1)/2
