解：
f(x+2) = (x+2) + g(x+2) = (x+2) + g(x) = (x + g(x)) + 2 = f(x) + 2
因此，f(x+8) = f(x) + 8。

因为 f(x) 在 [2, 4] 上的最大值为 1，
所以 f(x) 在 [10, 12] 上的最大值为 1 + 8 = 9。
