1. 设二次函数 $f(x) = ax^2 + bx + c (a < b < c)$ 的图象过点 $(1,0)$ 和 $B(m, -a)$。

(1) 若函数的顶点坐标为 $(-\frac{1}{4}, \frac{25}{16})$，求实数m的值；

(2)若函数图象的对称轴为x = x0,求x0的取值范围:

解：设f(x)+a与x轴交于(m, 0)
∴ am² + bm - b = 0
am² = -b(m - 1)
-b/2a = m²/(2m-2)
由(2)知, -1/2 < x₀ ≤ 0,
即 -1/2 < m²/(2m-2) ≤ 0
设 m²/(2m-2) = x, m² = (2m-2)x
∴ m² - 2mx + 2x = 0
m₁ = (2x - √(4x²-8x))/2, m₂ = (2x + √(4x²-8x))/2
∴ m₂ > m₁
令 m(x) = (2x + √(4x²-8x))/2
∴ dm/dx = 1 - (x-2)/√(4x²-8x)
(i) x₀ 为区间端点
x₀ = -1/2 或 x₀ = 0
(ii) x₀ 处 m'(x) 不存在,
即 1 - (x-2)/√(4x²-8x) 无意义
∴ x₀ ∈ (-1/2, 0] ∴ x₀ = 0
(iii) x₀ 处 m'(x) = 0
即 1 - (x-2)/√(4x²-8x) = 0
x₀ = 2(增根) 或 x₀ = -2/3
∴ x₀ ∈ (-1/2, 0] 舍
∴ x₀ = 0 或 -1/2 时取得局部极值
∴ x₀ = 0, m = 0; x₀ = -1/2, m = (√5-1)/2
∴ m < (√5-1)/2 ∴ k = (√5-1)/2
